## Cordova 설치
```
1 node.js 다운로드 설치 (npm 모듈 사용)
2 > node -v
3 > npm -v (cordova 설치)
4 > npm install cordova -g (cordova 프로젝트 생성)
5 > cordova -v (cordova 버전확인)
6 > D:\Kang\bitcamp104\Android\Cordova_Labs (폴더 생성)
7 > cordova create hello kr.or.bit.example HelloWorld -d (프로젝트 생성)
8 안드로이드용으로 사용하겠다. > cd hello (프로젝트 폴더에서)
9 > cordova platforms add android (or ios) (플랫폼 생성)
10 cordova 사용해서 빌드 APK 생성해서 (에뮬레이터) LOAD
11 에뮬레이터 실행 (안드로이드 스튜디오에서)
12 > cordova build android
13 > 로드 된 에뮬레이터에서 설치하기
14 > cordova run android
15 에뮬레이터 설치 완료
16 코드 수정 (build) 다시 배포 (build, run) 하면 다시 배포
17 안드로이드 스튜디오에서 open해서 편집 (Import CordovaProject_Android_2.docx 참고)
```
