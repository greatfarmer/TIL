# Linux

## 자주쓰는 명령어
- `df -h` 용량 확인
- `du -hs` 해당 디렉토리 용량 확인
- `netstat -tnlp` 네트워크 상태 확인
- `ps -ef | grep <디렉토리명>` 프로세스 상태 확인
- `rm -rf ./*` 현재 위치한 경로상의 폴더 및 파일 모두 삭제
- `history` 쉘 명령어 수행 목록 조회
- `last` 최근 접속 기록 및 최근 재부팅 기록 확인
